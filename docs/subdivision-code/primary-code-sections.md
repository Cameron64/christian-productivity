# Primary Code Sections

## [Title 30 - Austin/Travis County Subdivision Regulations](https://library.municode.com/tx/austin/codes/land_development_code)

**Primary subdivision authority**

**Location:** [Austin Land Development Code - Title 30](https://library.municode.com/tx/austin/codes/land_development_code)

### Key Chapters

#### [Chapter 30-5: Environment](https://library.municode.com/tx/austin/codes/land_development_code?nodeId=TIT30AUCO_CH30-5EN)
Covers drainage, water quality, and floodplain requirements.

**[Article 7: Requirements in All Watersheds](https://library.municode.com/tx/austin/codes/land_development_code?nodeId=TIT30AUCO_CH30-5EN_ART7REALWA)**
- **Division 1:** [Critical Water Quality Zone Restrictions (§30-5-261)](https://library.municode.com/tx/austin/codes/land_development_code?nodeId=TIT30AUCO_CH30-5EN_ART7REALWA_DIV1CRWAQUZORE_S30-5-261CRWAQUZO)

### Coverage

- Subdivision regulations in Austin's ETJ (Extra-Territorial Jurisdiction)
- Lot layout and design
- Street and infrastructure requirements
- Drainage and environmental protections

**Key Divisions:**

- Street design standards

## Quick Navigation

**For Subdivision Planning:**
- Start with [Title 30](https://library.municode.com/tx/austin/codes/land_development_code) for subdivision-specific requirements
- Reference [Chapter 30-5](https://library.municode.com/tx/austin/codes/land_development_code?nodeId=TIT30AUCO_CH30-5EN) for environmental/drainage issues

**For In-City Development:**
- Use [Title 25, Chapter 25-6](http://austin-tx.elaws.us/code/ldc_title25_ch25-6) for transportation/parking
