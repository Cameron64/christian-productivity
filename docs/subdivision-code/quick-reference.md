# Quick Reference Guide

## Key Topics Cross-Reference

### For Drainage Design

1. **Primary:** [Drainage Criteria Manual (DCM)](https://library.municode.com/tx/austin/codes/drainage_criteria_manual)
2. **Code:** [LDC Chapter 25-7 (Drainage)](https://services.austintexas.gov/edims/document.cfm?id=192466)
3. **Subdivision:** [Title 30, Chapter 30-5, Article 7 (Watersheds)](https://library.municode.com/tx/austin/codes/land_development_code?nodeId=TIT30AUCO_CH30-5EN_ART7REALWA)

**Common Tasks:**
- Storm drain sizing → [DCM Section 5](https://library.municode.com/tx/austin/codes/drainage_criteria_manual?nodeId=S5STDR)
- Floodplain regulations → [LDC §25-7-33](https://library.municode.com/tx/austin/codes/land_development_code?nodeId=TIT25LADE_CH25-7DR_ART2FLPR_S25-7-33FLPRALPECO), [§25-7-92](https://library.municode.com/tx/austin/codes/land_development_code?nodeId=TIT25LADE_CH25-7DR_ART3SPRE_S25-7-92ADREFAREAPLADEINFLHAZO)
- Erosion control → [DCM Section 1](https://library.municode.com/tx/austin/codes/drainage_criteria_manual?nodeId=S1DRPO)
- Drainage easements → [LDC §25-7-152](https://library.municode.com/tx/austin/codes/land_development_code?nodeId=TIT25LADE_CH25-7DR_ART4DRRE_DIV2DRIE_S25-7-152DREI)

**Common Tasks:**
- Street cross-sections → [TCM](https://library.municode.com/tx/austin/codes/transportation_criteria_manual)
- ROW widths → [TCM](https://library.municode.com/tx/austin/codes/transportation_criteria_manual)
- Intersection design → [TCM](https://library.municode.com/tx/austin/codes/transportation_criteria_manual)
- Pavement design → [TCM](https://library.municode.com/tx/austin/codes/transportation_criteria_manual)

### For Parking Requirements

1. **Primary:** [LDC Chapter 25-6, Article 7 (Off-Street Parking)](http://austin-tx.elaws.us/code/ldc_title25_ch25-6_art7of-stpalo)
2. **Reference:** [Transportation Criteria Manual (TCM)](https://library.municode.com/tx/austin/codes/transportation_criteria_manual)

**Key Provisions:**
- Minimum/maximum spaces → [LDC §25-6-475 (Division 1)](http://austin-tx.elaws.us/code/ldc_title25_ch25-6_art7of-stpalo_div1gere)
- On-street parking credits → [LDC Chapter 25-6, Article 7](http://austin-tx.elaws.us/code/ldc_title25_ch25-6_art7of-stpalo)
- Compact car provisions → Up to 30% for facilities with 12+ spaces
- CBD/DMU special rules → [LDC §25-6-591 (Division 5)](http://austin-tx.elaws.us/code/ldc_title25_ch25-6_art7of-stpalo_div5cbdmusppr)
