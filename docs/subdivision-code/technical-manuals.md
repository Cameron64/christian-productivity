## Using Technical Manuals

### When to Use Each Manual

**Drainage Criteria Manual:**
- Sizing storm drains and culverts
- Designing open channels
- Calculating drainage flows
- Planning erosion control

**Transportation Criteria Manual:**
- Determining street widths
- Planning right-of-way requirements
- Designing intersections
- Planning pedestrian/bicycle facilities

**Other Manuals:**
- Reference as needed for specific topics
- Often cross-referenced by DCM and TCM

### Manual Hierarchy

1. **Code sections** ([Title 25](https://library.municode.com/tx/austin/codes/land_development_code), [Title 30](https://library.municode.com/tx/austin/codes/land_development_code)) establish requirements
2. **Technical manuals** provide design standards and methods
3. **Rules** clarify specific interpretations

## Update Tracking

### How to Stay Current

1. **Subscribe to Code Updates:**
   - Check if Austin offers email notifications for code changes
   - Monitor City Council agendas for ordinance amendments

2. **Quarterly Review:**
   - Every 3 months, check Municode for updates to critical sections
   - Re-download any updated PDFs

3. **Version Control:**
   - Add effective date to PDF filenames: `DCM-Section-5-Storm-Drains-2024-01-15.pdf`
   - Keep previous version in archive folder if major changes
